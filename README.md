# typothunder

typothunder is a Realtime Multiplayer Typeracing Platform.

## Live Demo

[Typothunder](https://typothunder.vercel.app/)

## Features

- Typing with car animation
- Multiplayer Online TypeRacing
- Invite Friends via unique link
